# AI駆動開発 コラボレーションログ

本プロジェクトでは、複数のAI（Gemini/ChatGPT）をエンジニアおよびリスクマネージャーとして活用し、16時間想定の課題を短時間で高精度に完遂しました。

### AI活用のポイント
1. **アーキテクチャ設計:** クリーンアーキテクチャのレイヤー分離をAIに指示し、ドメイン中心のディレクトリ構造を自動生成。
2. **コンフリクト管理:** 複数AIの提案が衝突した際、人間がPdMとして最終判断を下し、一貫性を維持。
3. **トラブルシューティング:** 3Dエンジンのオクルージョン問題等、技術的な壁をAIと共に分析し、MVPにおけるスコープ判断（Known Issuesへの定義）を迅速に実施。
